Overview

WrapAction to wrap node within a block, class or module.

Note: if WrapAction is conflicted with another action (start and end are overlapped), we have to put those 2 actions into 2 within_file scopes.

def new_code
  "#{@code}\n#{' ' * @indent}" +
    NodeMutation.adapter.get_source(@node).split("\n").map { |line| "  #{line}" }
                .join("\n") +
    "\n#{' ' * @indent}end"
end
